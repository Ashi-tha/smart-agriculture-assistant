🌾 Smart Agriculture Assistant

An AI-powered web application that helps farmers make intelligent decisions using machine learning and real-time weather data.

🚀 Features
🌱 Crop Recommendation

Suggests best crops based on:

Soil nutrients (N, P, K)

Temperature

Humidity

pH

Rainfall

📊 Yield Prediction

Predicts expected crop yield using ML models

Helps in planning agricultural output

🌿 Plant Disease Detection

Upload plant leaf images

Detect diseases using deep learning

🌦 Weather Insights

Real-time weather data integration

Smart suggestions for irrigation and farming
